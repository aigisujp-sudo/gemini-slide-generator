Claude Code Skill

スライド全体を、
AIに1枚絵としてデザインさせる。

テーマや参考資料を渡すだけで、構成づくりからフルHDのスライドデザイン、 Google スライドへの組み立てまでを自動化する Claude Code スキル。

curl -fsSL https://gemini-slide-generator.netlify.app/install.sh | bash

~/.claude/skills/gemini-slide-generator にインストールされます。 利用には Gemini API キーと Google Slides / Drive の OAuth 設定が別途必要です(下記参照)。

What it does

できること

01

構成をAIが自動設計

テーマや参考資料(文章・PDF・Markdown)を渡すだけで、章立て・箇条書き・発表者ノートまで構成案を生成。

02

スライド全面を1枚のデザインとして生成

背景・レイアウト・文字を別々に組むのではなく、Geminiが完成デザインとして直接描く。

03

日本語タイトルも正確に描画

gemini-3-pro-image-preview を採用し、日本語の文字崩れを解消。長い見出しでも破綻しない。

04

Google スライドへ自動組み立て

生成画像をDrive経由でスライドに全面配置し、実行後すぐに編集用URLを発行する。

How to use

使い方

01

インストール

上のコマンドを実行するだけ。~/.claude/skills/ にスキルが追加される。

02

Claude Codeで依頼

「〇〇についてスライド作って」と伝えるだけ。構成案を確認しながら進む。

03

URLを受け取る

生成が終わると、そのまま開けるGoogle スライドの編集用URLが届く。

Requirements

必要なもの

利用開始前に、以下2つを用意してください。

  • GEMINI_API_KEY — Google AI Studio で取得したAPIキー。.env などで管理。
  • Google Cloud の OAuth クライアント(Slides / Drive 書き込み権限)。初回のみブラウザでの認可が必要。手順は references/google_setup.md を参照。
